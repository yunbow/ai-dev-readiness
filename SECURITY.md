# Security Policy

## Scope

This application is a static web application with no server-side processing or database. All answers and diagnosis results are processed entirely within the browser (LocalStorage / IndexedDB) and are never sent externally.

## Reporting a Vulnerability

If you discover a security concern (e.g., XSS, vulnerabilities in dependencies, data leakage risks in the sharing feature), please report it via [GitHub Issues](https://github.com/yunbow/ai-dev-readiness/issues).

If you'd rather not report sensitive details publicly, please contact the repository owner directly.

## Supported Versions

This project continuously maintains only the `main` branch. Older versions do not receive individual support.
