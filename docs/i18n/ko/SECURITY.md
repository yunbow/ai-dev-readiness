# 보안 정책

## 적용 범위

본 애플리케이션은 정적 웹 애플리케이션으로, 서버 사이드 처리나 데이터베이스를 사용하지 않습니다. 응답 내용과 진단 결과는 모두 브라우저 내(LocalStorage / IndexedDB)에서 처리되며, 외부로 전송되지 않습니다.

## 취약점 보고

보안 관련 우려 사항(XSS, 의존 패키지의 취약점, 공유 기능에서의 데이터 유출 위험 등)을 발견하신 경우 [GitHub Issues](https://github.com/yunbow/ai-dev-readiness/issues)를 통해 보고해 주세요.

민감한 내용을 공개된 장소에 보고하고 싶지 않은 경우, 저장소 소유자에게 직접 연락해 주세요.

## 지원 대상 버전

본 프로젝트는 `main` 브랜치만 지속적으로 유지 관리합니다. 이전 버전에 대한 개별 지원은 제공하지 않습니다.
