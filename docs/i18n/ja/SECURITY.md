# Security Policy

## 対象範囲

本アプリケーションは静的Webアプリケーションであり、サーバーサイドの処理やデータベースを持ちません。回答内容・診断結果はすべてブラウザ内(LocalStorage / IndexedDB)で処理され、外部に送信されることはありません。

## 脆弱性の報告

セキュリティ上の懸念事項(XSS、依存パッケージの脆弱性、共有機能でのデータ漏えいリスクなど)を発見した場合は、[GitHub Issues](https://github.com/yunbow/ai-dev-readiness/issues) からご報告ください。

機微な内容を公開の場で報告したくない場合は、リポジトリオーナーへ直接ご連絡ください。

## サポート対象バージョン

本プロジェクトは `main` ブランチのみを継続的にメンテナンスしています。過去バージョンへの個別対応は行っていません。
