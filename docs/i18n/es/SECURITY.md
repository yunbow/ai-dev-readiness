# Política de Seguridad

## Alcance

Esta aplicación es una aplicación web estática, sin procesamiento del lado del servidor ni base de datos. Todas las respuestas y resultados del diagnóstico se procesan íntegramente en el navegador (LocalStorage / IndexedDB) y nunca se envían al exterior.

## Reportar una vulnerabilidad

Si descubre un problema de seguridad (por ejemplo, XSS, vulnerabilidades en dependencias, riesgos de fuga de datos en la función de compartir), repórtelo a través de [GitHub Issues](https://github.com/yunbow/ai-dev-readiness/issues).

Si prefiere no reportar detalles sensibles públicamente, contacte directamente con el propietario del repositorio.

## Versiones compatibles

Este proyecto mantiene de forma continua únicamente la rama `main`. Las versiones anteriores no reciben soporte individual.
