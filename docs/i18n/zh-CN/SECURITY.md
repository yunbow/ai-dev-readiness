# 安全策略

## 适用范围

本应用程序是一个静态Web应用程序，不涉及服务器端处理或数据库。所有回答内容和诊断结果均仅在浏览器内(LocalStorage / IndexedDB)处理，不会发送到外部。

## 报告漏洞

如果您发现安全相关问题(例如 XSS、依赖包漏洞、分享功能中的数据泄露风险等)，请通过 [GitHub Issues](https://github.com/yunbow/ai-dev-readiness/issues) 进行报告。

如果您不希望在公开场合报告敏感内容，请直接联系仓库所有者。

## 支持的版本

本项目仅持续维护 `main` 分支，不对历史版本提供单独支持。
